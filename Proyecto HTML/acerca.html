Laboratorio HTML5

Demostración práctica de elementos y etiquetas estructurales poco comunes del estándar HTML5:

1. Elemento Anotación Ruby (<ruby>, <rt>)

Se utiliza para mostrar la pronunciación de caracteres de idiomas de Asia Oriental: 漢 かん 字 じ

2. Aislamiento y Dirección Bi-Direccional (<bdi>, <bdo>)

Aislamiento de texto con dirección desconocida (útil para nombres árabes o hebreos insertados dinámicamente): User_9921 (إياد) ha comprado un artículo.

Forzar direccionalidad inversa de texto completo: Este texto se leerá al revés.

3. Oportunidad de Salto de Línea (<wbr>)

Le indica al navegador de manera segura dónde se puede romper una cadena de texto extremadamente larga si falta espacio físico: unTextoDePruebaExtremadamenteLargoQueNoDebeRomperLaEstructuraContenedoraWeb.com

4. El elemento de datos estructurados (<data>)

Vincula una traducción legible por humanos con un valor para procesamiento interno de bases de datos:

  • Auriculares de Estudio
